01- Emmet

Work Flow - Merkup & Styling

Nesting ⬇️

example:

ul>li>a 🔂

result:

<ul> <li><a href=""></a></li> </ul>

elementos hermanos "+"

ejemplo:

section+header+nav🔂

result:

<section></section> <header></header> <nav></nav>

repetir elementos "*"

ejemplo:

ul>li*3🔂

result:

<ul> <li></li> <li></li> <li></li> </ul>

atributo clase

ejemplo:

div.contenedor🔂

result:

<div class="contenedor"></div>

multiples clases "."

ejemplo:

div.clase1.clase2.clasen🔂

result:

<div class="clase1 clase2 clasen"></div>

atributo identificador "#"

ejemplo:

nav#menu🔂

result:

<nav id="menu"></nav>

adionar texto "{}"

ejemplo:

div.{cristhiansantiago}🔂

result:

<div>cristhian santiago</div>

personalizar atributos "[]"

ejemplo:

a[href=www.google.com]🔂

result:

<a href="www.google.com"></a>

numeros consecutivos "$"

ejemplo:

ul>li.num$*3🔂

result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

direccion de numeros consecutivos "@"

ejemplo:

ul>li.num$@-*3🔂

result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

base numeros consecutivos "$"

ejemplo:

ul>li.num$@3*4🔂

result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

emmet - subir nivel

ejemplo:

section>article>div^p🔂

result:

<section> <article> <div></div> </article> <p></p> </section>

agrupar elementos "()"

ejemplo:

(header>nav)(section>aticle)🔂

result:

(header>nav)(section>aticle)

lorem ipsum dolor "lorem"

ejemplo:

lorem🔂
lorem20🔂
lorem*5🔂

result:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatem repellat, sapiente nihil dolore neque aliquid omnis dicta aperiam repudiandae a, illum accusamus sint. Quibusdam dolor cum sunt repudiandae iste tempore!

documents HTML "HTML:"

ejemplo:

html:xs🔂
html:xt🔂
html:4s🔂
html:4t🔂
html:5🔂

result:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

meta tags "meta:"

ejemplo:

meta:vp🔂
lorem20🔂
lorem*5🔂

result:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

reto html

ejemplo:

header#header>nav.navbar.navbar-inverse>div.container>(div.navbar-header>(button.navbar-toggle[type=button]>span.sr-only{Toggle navigation}+span.icon-bar*3)+a.navbar-brand[href="index.php"]>img[src="imgs/logo.png" alt=""])+div.collapse>ul.nav.navbar-nav>(li>a.link[href="index.html"]{Inicio})+(li.active.link>a[href="login.html"]{Ingresar})+(li>a.link[href="registro.html"]{Registrarse})+(li>a.link[href="contacto.html"]{Contacto})
